Accessible, progressive zooming for images.

## Installation
```bash
$ npm i @rpearce/image-zoom
```
or
```bash
$ yarn add @rpearce/image-zoom
```
or
```html
<!-- this build only needs React to be already present -->
<script async src="https://unpkg.com/@rpearce/image-zoom"></script>
```
